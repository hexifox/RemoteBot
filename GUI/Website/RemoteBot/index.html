What is RemoteBot?

Remote Bot is a prototype of a measurement station (Temperature, Humidity, Light, Pressure and others) communicating with a Host PC wirelessly (over ~40km) allowing manipulation and storage of the data transmitted in a graphical user interface (GUI) fitting the system. It would allow measurements in areas difficult to access, or with climate condition making long stay for human difficult. Its embedded wheel system will allow it to move on its own, following commands remotely. This system needs to be power efficient, show high rate and accurate samples and have reliable remote capabilities in order to meet the given requirements. These will be addressed through the choice of the components, such as a power-friendly microcontroller, solar panel, powerful RF modules and an array of strong sensors.

Since the start of its development, and because the milestones were reached faster than expected, it was decided to extend it with an online database, using MySQL, to make the data available at greater distance. It is now in fact online, and can be accessed from almost anywhere.

RaspBerry Pi2 B+

The Raspberry Pi 2 Model B is the second generation Raspberry Pi.

It replaced the original Raspberry Pi 1 Model B+ in February 2015.

Compared to the Raspberry Pi 1 it has:

  • A 900MHz quad-core ARM Cortex-A7 CPU
  • 1GB RAM
Like the (Pi 1) Model B+, it also has:
  • 4 USB ports
  • 40 GPIO pins
  • Full HDMI port
  • Ethernet port
  • Combined 3.5mm audio jack and composite video
  • Camera interface (CSI)
  • Display interface (DSI)
  • Micro SD card slot
  • VideoCore IV 3D graphics core

Because it has an ARMv7 processor, it can run the full range of ARM GNU/Linux distributions,

including Snappy Ubuntu Core, as well as Microsoft Windows 10 (see the blog for more information).

The Raspberry Pi 2 has an identical form factor to the previous (Pi 1) Model B+ and has complete compatibility with Raspberry Pi 1.

Sensors

The WDT, or Wireless Data transceiver, is the Remote Unit. It hosts all the sensoring section of the project.

It uses an RPi as its core, and take the following measurements:

  • Temperature, Pressure, Sea-Level pressure and Altitude, With a BMP 180 sensor.
  • Luminosity, with a simple LDR sensor.
  • Rain Drops, with a simple Rain Drop sensor.
  • Humidity and Temperature again, with a DHT AM2302 sensor.
  • Finally, Position and Time, using a GPS chip.

It also has special features, such as:

  • Camera interface and Streaming, over Wifi.
  • An on-board Touchscreen display will be implemented.
  • Xbee of course, for long-distance transmission.
  • A wheel system, to move around!

Both for the screen interface, and for all the analogue measurements, it uses Arduinos as slaves (an Uno and a Nano).
